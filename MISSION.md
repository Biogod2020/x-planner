# Mission

Build v0.1 of Mission Control Core: a local-first database core, MCP server, and agent skill pack for managing human goals with deterministic execution rules.

## Scope

- SQLite database
- Pydantic typed models
- Deterministic repository/state-machine logic
- Local MCP tools and resources
- Mission-control skill pack
- Pytest coverage for core rules

## Non-Goals

- Web UI
- Mobile app
- Telegram bot
- Email or calendar integration
- Notifications
- Multi-user authentication
- Cloud sync
- Full GitHub integration
- Automatic scheduling
